Introduction

C is an imperative, procedural language in the ALGOL tradition. It has a static type system. In C, all executable code is contained within subroutines (also called "functions", though not in the sense of functional programming). Function parameters are passed by value, although arrays are passed as pointers, i.e. the address of the first item in the array. Pass-by-reference is simulated in C by explicitly passing pointers to the thing being referenced.

C program source text is free-form code. Semicolons terminate statements, while curly braces are used to group statements into blocks.

The C programming language also exhibits the following characteristics when it comes to functions:

  • Function return values can be ignored, when not needed.
  • Function and data pointers permit ad hoc run-time polymorphism.
  • Functions may not be defined within the lexical scope of other functions.
  • Variables may be defined within a function, with scope.
  • A function may call itself, so recursion is supported.
Syntax

The syntax of C is very simple. The following rules apply:

  • #include <stdio.h> is an example line of code showing how to import libraries in C. This is usually done near the top of your file. If you are unsure what libraries are, please check the "Libraries" section of the documentation page.
  • A blank line is ignored by C. So it is used to organize the code and make it more readable.
  • Another thing that always appear in a C program, is main(). This is called a function. Any code inside its curly brackets {} will be executed.
  • printf() is a function used to output/print text to the console.
  • Every statement in C ends with a semicolon (e.g. printf(%d, number);).
  • return 0 ends the main function.

Strings

Strings in C are unique as they do not have their own specified data type. Instead, they are implemented as character arrays.

To declare a C-string, you must use the syntax listed:

char name[number];

The char is to declare a character type. The name can be any name of your choice and is to identify this specific type. The number can be any positive number and is meant to initialize an character array of the given number.

Each string's last index will contain a null-terminating character ('\0') so when initializing a string, make sure to add space for one more character in order to include the null-terminating character. Otherwise, undefined behavior can occur.

if you include the <string.h> library, you can use functions made specifically for strings like strcpy(), strcat(), strcmp(), etc.

Dynamic Programming

Dynamic programming is a technique used in computer science and mathematics to solve problems by breaking them down into smaller overlapping subproblems and solving each subproblem only once, storing the solutions to subproblems in a table to avoid redundant computations. This helps improve the efficiency of the algorithm. An example of this is recursion.

              
int fib(int n) {
  if (n <= 1) {
    return n;
  }
  return fib(n-1) + fib(n-2);
}

Dynamic programming allows programmers to break down tasks into smaller problems in order to solve for something bigger!

Linked Lists

A linked list is a linear data structure in which elements are stored in nodes, and each node points to the next node in the sequence. Unlike arrays, linked lists do not have a fixed size, and their elements (nodes) are not stored in contiguous memory locations. Instead, each node contains data and a reference (or link) to the next node in the sequence.

To create a node, you must create a new struct: 


struct Node {
  int data;
  Node* next;
}

You can then use these nodes to link data types together, kind of like a chain. The pros of using a linked-list is that it can be dynamically adjusted in terms of memory allocation. To allocate more memory for the linked list, you simply just create another node and have the previous node point to the new node. The cons of a linked-list is that search algorithms aren't made for these data types. It is always a linear search to traverse through a linked-list.
